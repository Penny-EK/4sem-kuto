## gennemgang af syntax

### Navngivning 
Mappenavne: snake_case

Funktionsnavne: camelCase

Komponenter: PascalCase med .jsx

Commits: commits bliver skrevet så kort og præcist som muligt. holdes helst som 2-3 ord som starter med "add", "change" eller "delete"

Branches: der bliver beskrevet hvad der arbejdes på efterfuldt med "-[navn]"
